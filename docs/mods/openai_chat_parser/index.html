Module contents¶

openai_chat_parser

A script to download, extract, and parse OpenAI chat conversation archives. The script takes a ZIP URL as input and processes the conversations, organizing them into separate folders with appropriate file formats based on the conversation content.

openai_chat_parser.__main__.download_zip(zip_url, destination_path)¶

Download a ZIP file from the given URL and save it to the specified destination path.

Parameters:

  • zip_url – The URL of the ZIP file.

  • destination_path – The folder where the ZIP file should be saved.

Returns:

The file path of the downloaded ZIP file.

Parse a conversation and save its content into separate files.

Parameters:

  • export_path – The folder where parsed conversations should be saved.

  • conv_num – The conversation number.

  • conversation – The conversation data as a dictionary.

openai_chat_parser.__main__.unzip_and_parse_conversations(zip_file_path, archive_path)¶

Unzip and parse the conversations from the ZIP file.

The function extracts the contents of the ZIP file, reads the ‘conversations.json’ file, reverses the conversations order, and then calls the parse_conversation function for each conversation to parse and save its content.

Parameters:

zip_file_path – The file path of the ZIP file containing the conversations.
